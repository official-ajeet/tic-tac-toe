# Tic-Tac-Toe Game

This repository holds the code for a classic Tic-Tac-Toe game where two players can compete on the same computer!

## Features

- Play Tic-Tac-Toe with a friend locally.
- Simple and clean user interface.
- Uses HTML, CSS, and JavaScript for a web-based experience.

## How to Play

1. Clone this repository or download the files.
2. Open the index.html file in a web browser.
3. Two players take turns clicking on the cells to make their moves (X and O).

## Technologies Used

- HTML: Defines the basic structure of the game board and cells.
- CSS: Styles the game board and individual cells for visual appeal.
- JavaScript: Handles game logic, player interaction, turn tracking, and win conditions.
